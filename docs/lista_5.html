üìä Interpretando Raz√µes de Preval√™ncia

Pseudo R¬≤

O que √© Pseudo R¬≤?

An√°logo ao R¬≤ para modelos log√≠sticos/Poisson. Indica ‚Äúpropor√ß√£o de verossimilhan√ßa explicada‚Äù.

M√©todos comuns: - Hosmer-Lemeshow: Baseado em deviance - Cox-Snell: Ajustado pelo tamanho amostral - Nagelkerke: Normalizado para variar 0-1

# Fun√ß√£o personalizada
logisticPseudoR2s <- function(LogModel) {
  dev <- LogModel$deviance
  nullDev <- LogModel$null.deviance
  modelN <- length(LogModel$fitted.values)
  
  R.l <- 1 - dev / nullDev
  R.cs <- 1 - exp(-(nullDev - dev) / modelN)
  R.n <- R.cs / (1 - (exp(-(nullDev / modelN))))
  
  data.frame(
    Metodo = c("Hosmer-Lemeshow", "Cox-Snell", "Nagelkerke"),
    Pseudo_R2 = c(round(R.l, 3), round(R.cs, 3), round(R.n, 3))
  )
}
